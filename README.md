# Epic7_Shop_Refresh
> Epic7_Shop_Refresh

[Epic 7](https://epic7.smilegatemegaport.com/) secret shop auto buy, looking mainly for Mystic and Covenant Bookmarks.

## Instalation

```bash
pip install -r requirements.txt
```

## Usage

Needs to be used in a 1080p screen

```bash
python main.py
```
